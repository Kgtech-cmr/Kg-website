Overview of KERM MD V2 WhatsApp Bot

KERM MD V2 is a powerful WhatsApp bot with over 140+ plugins, designed to automate tasks, provide information, and enhance user engagement. Below is a breakdown of its commands and features:

Converter Commands
  • FANCY - Create fancy text
  • IMG - Convert media to image
  • MP3 - Convert media to audio
  • MP4 - Convert media to video
  • PHOTO - Take a screenshot or capture an image
  • STICKER - Create a WhatsApp sticker
  • TAKE - Take or download content
Download Commands
  • APK - Download APK files
  • FB - Download Facebook videos
  • INSTA - Download Instagram media
  • MEDIAFIRE - Download from Mediafire
  • PLAY - Download from Google Play
  • SPOTIFY - Download music from Spotify
  • TWITTER - Download Twitter videos
  • VIDEO - Download video content
  • YTA - YouTube audio download
  • YTV - YouTube video download
KERM AI Commands
  • ANIMEGEN - Generate anime-related content
  • AOYO - AI-powered image generation
  • BINGAI - Microsoft Bing AI integration
  • BLACKBOX - AI-powered code generation
  • DALLE - Generate images using DALL-E
  • ENHANCE - Enhance image quality
  • GPT - Chat with OpenAI's GPT
  • GPT2 - Use OpenAI's GPT-2 for text generation
Group Management Commands
  • ACCEPT - Accept group invitations
  • ADD - Add a user to the group
  • ADMINS - Manage group admins
  • ANTIBOT - Enable anti-bot protection
  • ANTILINK - Prevent users from posting links
  • ANTIDEMOTE - Prevent users from demoting others
  • ANTIPROMOTE - Prevent users from promoting others
  • COMMON - Manage common group settings
  • DEL - Delete a message
  • DEMOTE - Demote a user from admin
  • DIFF - Show differences between group versions
  • GDESC - Set group description
  • GINFO - Display group information
  • GJID - Get group JID
  • GNAME - Set group name
  • GOODBYE - Set goodbye messages for members leaving
  • GPP - Set group profile picture
  • INVITE - Invite a user to the group
  • KICK - Remove a user from the group
  • LEAVE - Leave the group
  • MUTE - Mute the group
  • PROMOTE - Promote a user to admin
  • REJECT - Reject group invitation
  • REQUESTS - View pending group requests
  • TAG - Tag a user
  • TAGALL - Tag all group members
  • UNMUTE - Unmute the group
  • VOTE - Create a voting poll
  • WELCOME - Set welcome messages for new members
System Commands
  • CHECKUPDATES - Check for new updates
  • CPU - Display system CPU usage
  • DISABLE - Disable a plugin or feature
  • ENABLE - Enable a plugin or feature
  • KERM - Manage KERM bot operations
  • INSTALL - Install new plugins or features
  • LOGOUT - Logout the bot
  • PATCH - Apply patches to the bot
  • PING - Check the bot's response time
  • PLUGIN - Manage bot plugins
  • REMOVE - Remove a plugin or feature
  • RESTART - Restart the bot
  • RUNTIME - Display the bot's runtime
  • SHUTDOWN - Shut down the bot
  • UPDATE - Update the bot to the latest version
Tools Commands
  • BITLY - Shorten a URL using Bit.ly
  • FULLSS - Capture a full-page screenshot
  • QR - Generate a QR code
  • RMBG - Remove background from an image
  • SHORTLINK - Shorten a URL using another service
  • TIME - Display the current time
  • TINYURL - Shorten a URL using TinyURL
  • TRIM - Trim a video or audio file
  • UPLOAD - Upload media or files
  • WM - Add a watermark to an image
WhatsApp Commands
  • ARCHIVE - Archive a chat
  • BLOCK - Block a user
  • JID - Get a contact's JID
  • MYPP - Change your profile picture
  • PIN - Pin a chat
  • READ - Mark a chat as read
